## 🎯 Overview

A **graphical calculator** built in **C++ using EasyX**, evolved from a console-based application into a fully interactive GUI.

This project demonstrates:
- Transition from CLI → GUI
- Event-driven programming
- Real-time rendering loop

---

## ✨ Features

✅ Basic operations: `+  -  *  /`  
✅ Interactive GUI with buttons  
✅ Mouse input handling 🖱️  
✅ Operation history (last 5 calculations)  
✅ Clean modular logic (reused from console version)
