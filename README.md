# 📟 Calculator (C++)

A command-line calculator application developed in C++ that performs basic arithmetic operations while maintaining a history of all calculations within the session. The project demonstrates structured programming, modular design, and dynamic data handling using the Standard Template Library (STL).

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division with error handling (division by zero)  
- 📜 Calculation history tracking during runtime  
- 🔁 Menu-driven interface for repeated operations  

---

## 🧠 Technologies Used

- C++
- Standard Template Library (STL)
  - `vector`
  - `string`
- Basic mathematical operations

---

## 🏗️ System Design

The application is structured using:
- **Modular functions** for arithmetic operations  
- A **dynamic vector container** to store calculation history  
- A **loop-based menu system** for continuous user interaction  

This design ensures readability, scalability, and maintainability.
